# BirdCLEF+ 2026 中文运行与提交指南

这一版已经不是旧的单路 log-mel baseline。当前推荐方案是：

- `logmel_pcen` 双通道频谱
- `attn` 事件感知池化
- ConvNeXt/EfficientNet backbone
- `train_soundscapes` 作为域内训练数据
- rare-class balanced sampler
- 推理阶段 taxonomy smoothing、temporal smoothing
- 可选 Perch/BirdNET/其他公开模型 sidecar CSV 做 rank-space 修正

## 1. 准备数据

```powershell
cd D:\kaggle\birdclef2026
python scripts/download_data.py --out data/birdclef-2026 --copy
python scripts/prepare_metadata.py --data-dir data/birdclef-2026 --out-dir data/processed
python scripts/check_gpu.py
```

## 2. 下载模型资产

服务器训练推荐先下载当前最强 anchor 的预训练 backbone，同时准备 Google Perch v2 CPU 这个外部 Kaggle 模型：

```powershell
python scripts/download_model.py --preset anchor_v2_strong
```

会生成：

```text
models/pretrained/convnext_base.fb_in22k_ft_in1k.pth
models/kaggle/...
models/model_assets.json
```

Perch v2 CPU 是 TensorFlow/Kaggle Model，不是 PyTorch backbone。它适合作为 teacher、embedding 提取器或 sidecar CSV 来源，不能直接传给 `train.py --pretrained-path`。

如果只想下载 PyTorch anchor，不下载 Kaggle 外部模型：

```powershell
python scripts/download_model.py --preset anchor_v2_strong --skip-kaggle
```

## 3. 推荐训练命令

16G 显存服务器直接用 ConvNeXt-Base anchor：

```powershell
python train.py --data-dir data/birdclef-2026 --meta-dir data/processed --out-dir outputs/anchor_v2_convnext_base_fold0 --model convnext_base.fb_in22k_ft_in1k --pretrained-path models/pretrained/convnext_base.fb_in22k_ft_in1k.pth --epochs 20 --fold 0 --batch-size 4 --grad-accum 4 --duration 10 --channels-last --include-soundscapes --spec-mode logmel_pcen --pooling attn --head-hidden 768 --drop-path 0.2 --balanced-sampler --mixup-alpha 0.3 --mixup-p 0.5 --spec-augment-p 0.5 --scheduler cosine --lr 1e-4 --weight-decay 1e-4
```

训练 5 折：

```powershell
0..4 | ForEach-Object {
  python train.py --data-dir data/birdclef-2026 --meta-dir data/processed --out-dir outputs/anchor_v2_convnext_base_fold$_ --model convnext_base.fb_in22k_ft_in1k --pretrained-path models/pretrained/convnext_base.fb_in22k_ft_in1k.pth --epochs 20 --fold $_ --batch-size 4 --grad-accum 4 --duration 10 --channels-last --include-soundscapes --spec-mode logmel_pcen --pooling attn --head-hidden 768 --drop-path 0.2 --balanced-sampler --mixup-alpha 0.3 --mixup-p 0.5 --spec-augment-p 0.5 --scheduler cosine --lr 1e-4 --weight-decay 1e-4
}
```

如果显存还有余量，可以把 `--batch-size 4 --grad-accum 4` 改成 `--batch-size 8 --grad-accum 2`。

如果显存吃紧，使用小模型：

```powershell
python train.py --data-dir data/birdclef-2026 --meta-dir data/processed --out-dir outputs/anchor_v2_small_fold0 --model tf_efficientnet_b0_ns --epochs 10 --fold 0 --batch-size 8 --grad-accum 2 --duration 8 --channels-last --include-soundscapes --spec-mode logmel_pcen --pooling attn --head-hidden 256 --balanced-sampler
```

还不够就继续降：

```powershell
--batch-size 4 --grad-accum 4 --duration 5
```

## 4. 本地推理

只用 anchor：

```powershell
python infer.py --data-dir data/birdclef-2026 --checkpoint outputs/anchor_v2_convnext_base_fold0/fold0_best.pt --out submission.csv --tax-genus-alpha 0.15 --tax-class-alpha 0.05 --temporal-smooth-alpha 0.15
```

多 fold ensemble：

```powershell
python infer.py --data-dir data/birdclef-2026 --checkpoint outputs/anchor_v2_convnext_base_fold0/fold0_best.pt --checkpoint outputs/anchor_v2_convnext_base_fold1/fold1_best.pt --out submission.csv --tax-genus-alpha 0.15 --tax-class-alpha 0.05 --temporal-smooth-alpha 0.15
```

如果你有 Perch、BirdNET 或公开 notebook 生成的 sidecar submission CSV：

```powershell
python infer.py --data-dir data/birdclef-2026 --checkpoint outputs/anchor_v2_convnext_base_fold0/fold0_best.pt --out submission.csv --sidecar-csv subm_birdnet_v24.csv --sidecar-weight 0.03 --sidecar-rank-blend --sidecar-topk 48 --sidecar-budget 0.006 --tax-genus-alpha 0.15 --tax-class-alpha 0.05 --temporal-smooth-alpha 0.15
```

sidecar CSV 必须包含 `row_id` 和与 `sample_submission.csv` 一致的物种列。

## 5. Kaggle 提交

上传到 Kaggle Dataset：

- `kaggle_submission_standalone.py`
- `fold0_best.pt`
- 可选：`fold1_best.pt`、`fold2_best.pt`
- 可选：sidecar CSV

Kaggle Notebook 添加两个 Input：

- 官方比赛数据 `birdclef-2026`
- 你的权重 Dataset

设置：

- Accelerator: None
- Internet: Off

运行：

```python
!python /kaggle/input/birdclef2026-weights/kaggle_submission_standalone.py
```

如果自动找权重不准，编辑 `kaggle_submission_standalone.py` 顶部：

```python
CHECKPOINT_PATHS = [
    "/kaggle/input/birdclef2026-weights/fold0_best.pt",
]
```

如果要加 sidecar：

```python
SIDECAR_CSV_PATHS = [
    "/kaggle/input/birdclef2026-weights/subm_birdnet_v24.csv",
]
SIDECAR_WEIGHTS = [0.03]
```
